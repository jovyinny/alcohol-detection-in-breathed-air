# alcohol-detection-in-breathed-air
alcohol detection in breathed air

**The components used in the projects**

  - MQ2 sensor
  - Arduino Nano
  - LEDs(RGB)
  - LCD(I2C LCD)
  - 9V battery
  - jumper wires
  
  *Arduino code and circuit diagram are in the master branch*
